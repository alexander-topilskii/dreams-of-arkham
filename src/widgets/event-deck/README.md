# Event deck

Виджет отображает колоду событий и управляет её состоянием: стопкой закрытых карт, списком открытых событий и сбросом. Конфигурация поступает из JSON-файла с описанием карт и границами добора за одно событие.

## Использование
- Импортируйте `EventDeck` и конфигурацию: `import deckSource from "../../data/event-deck.json";`
- Создайте экземпляр: `new EventDeck(container, deckSource as EventDeckConfig)`.
- Вызовите `triggerEvent()` для моделирования игрового события (вытягивает 0+ карт) и `reshuffleDiscard()` для перемешивания сброса и возврата в стопку.

## Поведение
- Колода отображается как стопка карт рубашкой вверх; счётчик показывает количество оставшихся карт.
- Открытые события выводятся вертикальным списком с прокруткой. Клик по карточке переносит её в сброс.
- Секция сброса показывает собранные события и служит источником для перемешивания обратно в стопку.

## Завершение
Для удаления виджета достаточно убрать его корневой элемент из DOM — дополнительных ресурсов он не удерживает.
