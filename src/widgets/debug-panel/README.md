## Name
DebugPanel

## Purpose
DebugPanel предоставляет компактную панель отладки для ручного изменения игровых состояний и вызова служебных действий. Виджет предназначен для размещения в правой колонке рядом с игровым движком и позволяет сгруппировать контролы для победных, пораженческих и иных параметров.

## Visual Overview
```
+---------------------------------------------+
| Debug панель                                |
|---------------------------------------------|
| ПАРАМЕТРЫ ПОБЕДЫ                            |
| [Собранные улики] [-] [ 2 ] [+]             |
| [Сегменты ритуала] [-] [ 1 ] [+]            |
| [Финальная печать] [-] [Да] [+]             |
|                                             |
| ПАРАМЕТРЫ ПОРАЖЕНИЯ                         |
| [Очки гибели]      [-] [ 0 ] [+]            |
| [Активность культа][-] [ 3 ] [+]            |
| [Пробуждение]     [-] [Нет] [+]             |
|---------------------------------------------|
| КАРТЫ                                        |
| [Добавить карту] [Показать CardHand]         |
+---------------------------------------------+
```
- Фон: полупрозрачный тёмно-синий `rgba(15, 23, 42, 0.9)`.
- Отступы: 16px внутри панели, 12px вертикальный промежуток между группами.
- Шрифт: `system-ui`, размер текста 13px, заголовок панели 14px, названия групп 11px uppercase.
- Кнопки используют стиль из `createDebugButton` с плавным наведением.

## Behavior
- Конструктор создаёт корневой контейнер с указанными стилями и при необходимости добавляет заголовок.
- Метод `addGroup` добавляет вертикальный стек с заголовком раздела и возвращает контейнер для контролов.
- `addNumericControl` создаёт строчку с подписью и кнопками уменьшения/увеличения значения с учётом `min/max/step` и вызывает `onChange` после применения.
- `addBooleanControl` создаёт строчку с подписью и кнопками переключения булева состояния, вызывая `onChange` после изменения.
- Все методы автоматически добавляют созданные элементы в панель или указанную группу и возвращают созданный элемент для дополнительной настройки при необходимости.

## API (Props / Inputs / Outputs)
| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `constructor(container, options?)` | `(HTMLElement \| null \| undefined, DebugPanelOptions?) -> DebugPanel` | — | Создаёт панель и, если передан контейнер, монтирует её в DOM. |
| `options.title` | `string` | — | Текст заголовка панели. |
| `addGroup` | `(title: string) -> HTMLDivElement` | — | Создаёт группу контролов с заголовком и добавляет её в панель. |
| `addNumericControl` | `(group: HTMLElement, label: string, config: DebugNumericControlConfig) -> HTMLDivElement` | — | Добавляет числовой контрол с кнопками ±. |
| `addBooleanControl` | `(group: HTMLElement, label: string, config: DebugBooleanControlConfig) -> HTMLDivElement` | — | Добавляет булев переключатель на кнопках ±. |
| `element` | `HTMLDivElement` (getter) | — | Корневой DOM-элемент панели для прямого доступа. |

## States and Examples
- **Пустая панель**: создана без групп — отображается только заголовок (если указан).
- **Стандартная панель**: несколько групп с числовыми и булевыми контролами; значения обновляются мгновенно при нажатии кнопок.
- **Границы значений**: `addNumericControl` ограничивает значение в пределах `min`/`max`, предотвращая выход за диапазон.
- **Булевы переключатели**: отображают `Да`/`Нет` в зависимости от возвращаемого значения `config.get()`.

## Lifecycle
- Инициализация происходит в конструкторе: создаётся DOM и при необходимости монтируется в переданный контейнер.
- Обновление значений осуществляется через функции `config.get()` и `config.set()` переданные при создании контролов.
- Очистка: при удалении панели из DOM дополнительных действий не требуется; слушатели событий привязаны к кнопкам и удаляются вместе с элементами.
- Зависимости: модуль `../debug/debug` (функция `createDebugButton`), стили встраиваются инлайново.

## Integration Example
```ts
import { DebugPanel } from "../widgets/debug-panel/debug-panel";

const panel = new DebugPanel(document.getElementById("debug-root"), { title: "Debug панель" });
const victoryGroup = panel.addGroup("Параметры победы");
panel.addNumericControl(victoryGroup, "Собранные улики", {
    get: () => state.victory.collectedClues,
    set: (value) => { state.victory.collectedClues = value; },
    min: 0,
    onChange: syncVictory,
});
```
